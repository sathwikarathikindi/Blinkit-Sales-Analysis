# Blinkit-Sales-Analysis
My First Power BI Data Analytics Project

This project analyzes Blinkit sales and operational performance using an interactive Power BI dashboard.
It highlights key KPIs such as Total Sales, Average Sales, Number of Items, Fat Content distribution, Item Types, Outlet Size, Location, and Outlet Establishment trends.
All visuals are based on the Blinkit dataset.
Project Objective


The goal of this project is to:

Understand sales patterns across outlets

Compare item types and fat content impact

Analyze outlet performance by size, type, and location

Visualize sales growth over the years

Practice Power BI data modeling and visualization techniques
